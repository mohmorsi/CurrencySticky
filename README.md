# CurrencySticky 💲📌

A lightweight macOS Swift program that displays the live USD to CAD exchange rate in a pinned, borderless sticky note. 

Built entirely with native Swift (no heavy third-party software or API keys required), this script creates a classic yellow sticky note that hovers over your other windows and updates automatically.

---

## **Features**
*   **Always on Top:** The sticky note floats above all other application windows.
*   **Live Data:** Fetches the current USD to CAD exchange rate using a free public API.
*   **Auto-Updating:** Automatically refreshes the rate every hour.
*   **Native & Lightweight:** Uses macOS `Cocoa` and `Foundation` libraries. Zero external dependencies.
*   **Drag & Drop:** Click and drag the sticky note anywhere on your screen.

---

## **Prerequisites**
You only need a Mac. The Swift compiler comes pre-installed with Xcode Command Line Tools. If you haven't installed them yet, macOS will prompt you to do so when you try to run the script.
